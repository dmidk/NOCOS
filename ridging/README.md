Installation
#############

Software requirements:
Python 3.9 with following packages:
  - numpy
  - xarray (including dask, netCDF4 and bottleneck)
  - pyyaml

Use the provided file conda-env_NOCOS-RIDGING.yml to install a suitable conda environment.
